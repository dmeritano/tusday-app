TusDay -- Project & Tasks Administration App

PROJECT FEATURES

  • Backend

    • Node JS
    • Express
    • MVC Architecture - (View => React in this solution)
    • MongoDB (Atlas)
    • ORM: Mongoose
    • Security:
      • Authentication and Authorization
      • Public and protected endpoints
      • Custom security middleware
      • Password hashing with Bcrypt
      • JWT
      • Single use tokens
      • CORS (Allowing Frontend App URL)
    • Nodemailer
    • Socket.io:
      • Server
      • Rooms
      • Listen and Emit messages
    • Repository: Backend

  • Frontend

    • React JS
    • React Router Dom
    • Public and protected routes
    • Context API
    • Providers - Hooks
    • Layouts, Pages, Components
    • Axios
    • Tailwind CSS
    • HeroIcons, Headless UI
    • Repository: Frontend
